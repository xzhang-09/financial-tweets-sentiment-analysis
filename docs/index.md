## Overview

This project demonstrates how to build an end-to-end machine learning system for financial tweet sentiment analysis. The system classifies financial tweets into three sentiment categories: bullish, neutral, and bearish.

- **Project**: Financial Tweets Sentiment Analysis
- **Model**: Logistic Regression baseline and Transformer fine-tuning
- **Framework**: PyTorch, scikit-learn and FastAPI
